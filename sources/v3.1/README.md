# v3.1 data

Vendored from https://github.com/Cambridge-Sustainable-Computing-Lab/Green-Algorithms-data
(the upstream Green Algorithms data repository), v3.1 directory, fetched 2026-06-13.

- Carbon intensity: Electricity Maps 2024 yearly life-cycle data (ODbL licence,
  see carbon-intensity/electricitymap/LICENSE).
- Chips: manually curated CPU/GPU lists (chips/manual).
- `site/data.js` is generated from these files by `scripts/build_data.py`.
